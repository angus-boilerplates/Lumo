# Lumo

Lumo is a simple boilerplate for building Laravel applications. It comes with...

- Laravel 11
- TailwindCSS
- Livewire
- Breeze authentication
- FlyonUI components

## Installation

Clone the project

```bash
git clone git@github.com:angus-boilerplates/Lumo.git
```

Go to the project directory

```bash
cd Lumo
```
